# Enterprise-Incident-Response-Plan

This repository contains a comprehensive Incident Response Plan (IRP) designed for a hybrid university infrastructure.

## Key Features:
- **Framework:** Aligned with NIST SP 800-61 Rev. 2.
- **Scope:** Covers Ransomware, Phishing, and Unauthorized Access.
- **Compliance:** Integrates FERPA and GLBA regulatory requirements.
- **Deliverables:** Includes technical playbooks, severity scoring, and communication protocols.
